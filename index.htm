git init  (( Git-Obtendo-um-Repositório | cria um novo subdiretório chamado .git ))
git add *.html  (( agrupa os arquivos p o commit ))
git commit -m "first commit"  (( primeiro commit ))
git remote add origin git@github.com:dj79/git-windows.git  (( se conecta a origin para iniciar de arquivos ))
git push -u origin master  (( envia e atualiza o servidor com os código-fonte ))
  • Instalar o Git no Windows
  • Abir o prompt do Git (ctrl + ' chama o terminal no VS-Code)
  • Acessar a pasta desejada p/ via prompt e excutar os camandos acima
  git status  (( verifica o branch master, listando Untracked files e Changes to be committed ))
  git add f1.htm f2.css f3.js  (( add-multiple-files ))
  git commit -m 'mensagem de lembranca do commit' (( somente p/ Changes to be committed))
  git push  (( envia de fato p/ o servidor ))

Instalar TortoiseGit, ferramenta Git Client, p/ organizar commando básicos e configurações.
Clique botão direito no Explorer Windows, TortoiseGit, Settings.
